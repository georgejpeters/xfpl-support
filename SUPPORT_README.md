# xFPL Support

Thanks for testing xFPL. If you run into bugs or have feedback, please send it here.

## Contact
- GitHub Issues: https://github.com/georgejpeters/xfpl-support/issues
- Email: support@yourdomain.com

When reporting a problem, please include:
- Device model + iOS version
- App version/build (from Settings → xFPL)
- Steps to reproduce
- Screenshots or screen recording (if possible)

## FAQ
**The app won’t load data**
- Check your internet connection.
- Make sure you’re on the latest TestFlight build.
- If it still fails, report it with the error message or screenshot.

**Expected points (xP) is missing**
- It may be recalculating. Try again in a few minutes and report if it persists.

## Privacy
This app is in testing. If you want your data removed, contact us.
